Logstash filter tester

A simple way to test your Logstash's configurations files

The goal of this site is to provide you a Logstash instance to test the parsing of a log file.
 
It was and will never be mean to read or write to an external source, such as Elasticsearch, Redis, Kafka, etc.
 
Each time you launch the processing of your configuration, your full configuration will be saved in local, which mean that they will be persistant between your sessions !
 
You wil be able to :
- Enter some extra inputs fields, such as type, pilote, filepath, etc., depending of your original real input
- Enter some representative data from the original logs files
- Build your logstash configuration from scratch in the embedded editor
 
Technical considerations:
  • Logstash if package as a light version, with a limited number of inputs and outputs ; and is optimized to run faster than usual instances
  • Logstash is version 5.6.15, with all it's plugins updated to latest version
  • Each Logstash job is launch with 1Go of memory

As it is currently in beta, if you encounter a bug, need a new functionnality or have a specific need, feel free to contact me at this mail address to talk about it.
